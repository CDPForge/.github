# CDPForge

> **CDPForge** is a proprietary, modular, and scalable Customer Data Platform (CDP) to collect, unify, govern, and activate customer data in real time.

## Key Features
- **Modular architecture**: Lightweight core with native support for plugins, built around an Apache Pulsar bus.
- **Multi-channel collection**: Ingest events from websites, apps, and APIs, or via file imports.
- **Identity resolution**: Merge anonymous and identified sessions into a single unified profile (Person).
- **Audience segmentation**: Build segments with a visual query builder and preview matching profiles.
- **Real-time enrichment**: A plugin pipeline enriches every event in real time (geo, identity, behavioral scoring).
- **Triggers & personalization**: Automate event-driven actions and serve targeted on-site content to qualified visitors.
- **Consent management**: A native consent manager and preference center with a per-client purpose catalog, consent ledger, and ingestion gating.
- **Governance & GDPR**: Audit logs and tooling for GDPR deletion and data export requests.
- **AI assistant**: Query your data in natural language, with a configurable LLM provider per client.
- **Extensible design**: SDK and REST APIs for building and integrating custom modules and plugins.
- **Kubernetes-native deployment**: Modular microservices deployable and scalable independently via Helm.

## Goal
Provide a powerful yet easy-to-operate solution for both startups and large enterprises, with maximum flexibility for evolving data collection, customer profiling, and activation strategies.
